Step 1 — Load your music

Click Analyze in the top-right corner. Four ways to bring in your library:

Scan a folder

Set your Music Folder in Settings. Desktop apps perform deep local audio analysis for BPM and Key.

Mac + Windows

Apple Music

Click Analyze → Apple Music and pick a playlist. Uses your existing macOS library metadata.

Mac only

M3U / TXT file

Works with playlist exports from Rekordbox, Serato, and other DJ software.

All Platforms

Rekordbox XML

Export collection as XML from Rekordbox, then import into DJFriend to sync your library.

Mac + Windows
Note on Web Analysis:
To protect your privacy and bandwidth, the Web App performs AI metadata estimation powered by Groq. A Groq API Key is required in settings for this feature. For direct audio file analysis, use the Desktop versions.

Steps 2–4 — Build your set

2

Shape your energy arc

The Energy Curve editor shows 5 draggable handles. Use presets like Build-up, Peak, or W-shape.

3

Set duration and filters

Choose runtime (30–180 min) and narrow by genre, mood, or venue type.

4

Generate and refine

DJFriend builds a tracklist. Drag rows to reorder or use swap icons for alternatives.

Step 5 — Export your set

Save as M3U

Load the playlist in Rekordbox, Serato, or any other DJ software.

Spotify Playlist

Authorize your account to create a new playlist automatically.
