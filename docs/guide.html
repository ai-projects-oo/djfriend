DJ Guide

How to use DJFriend

Everything you need to go from an empty library to a polished, export-ready DJ set — step by step.

01

Getting started

DJFriend is a desktop app for Mac and Windows (and a web version at djfriend.onrender.com). Download the right build for your machine from the Downloads page and install it.

On first launch the app opens to the Library tab. All analysis happens locally — your music never leaves your computer.

First time? Head to Settings (gear icon, top right) and add your Spotify Client ID and Secret. This enables genre data for every track. Without it the app still works — you'll just have fewer genre signals.
02

Load your library

Go to the Library tab and choose your source. DJFriend will scan, analyse, and cache the results in results.json in the same folder — subsequent loads are instant.

Local folder

Click Choose folder and point to any directory. Sub-folders are scanned recursively. Supported: MP3, FLAC, AAC, WAV, M4A, OGG.

Apple Music

Mac only. Pick any playlist from your Apple Music library. DJFriend reads the local files directly — no export needed.

Rekordbox XML

Export your collection from Rekordbox (File → Export Collection as XML), then import the file. BPM and key tags carry over automatically.

M3U / TXT

Drop in any M3U, M3U8, or plain text file (one track per line). Works on the web version too, without audio files.

BPM doubling? If a track has an existing BPM tag, DJFriend reads it and uses it as a hint during analysis — this prevents the common ×2 detection error on slow tracks. If a track still shows a doubled BPM after loading, open the track editor and use the ÷2 button to halve it.
03

Import a Spotify playlist

Go to the Import tab. Paste a Spotify playlist URL (or the raw playlist ID) and click Import. DJFriend finds the matching tracks in your local library and shows you what was found.

1
Copy the playlist URL

In Spotify: right-click the playlist → Share → Copy link to playlist. A full URL like https://open.spotify.com/playlist/37i9d… or just the 22-character ID both work.

2
Paste and import

Paste into the input field and click Import Playlist. Matched tracks are shown with a green checkmark; unmatched ones appear greyed out.

3
Use it in the generator

Once imported, the playlist is available in the Source card in the Generator sidebar. Select it to restrict set generation to only those tracks (see Generate from playlist).

04

Energy curve

The energy curve defines the vibe arc of your set — how it rises, peaks, and falls. Draw it before generating.

Presets

Five presets cover the most common set shapes:

  • Warm-up — gentle ramp from low to mid energy
  • Peak-time — sustained high energy throughout
  • Festival arc — slow build to a sharp peak, then gradual decline
  • Closing — starts mid, peaks, then winds down
  • Roller-coaster — alternating highs and lows for variety

Drawing freehand

Click anywhere on the curve editor canvas to add a control point, then drag it. Right-click a point to remove it.

Tip: fewer control points = smoother curve = more musical transitions. Start with 3–4 points and add more only if you need a specific shape.
05

Preferences

The generator sidebar contains several controls that shape which tracks are eligible:

Venue & Set Phase

Choose your venue (Club, Bar, Wedding, Festival, Private event) and phase (Warm-up, Peak time, Cool-down, After-party). DJFriend will favour tracks that suit the vibe.

BPM Range

Set a minimum and maximum BPM. Only tracks within this range will be used.

Genre Filter

Pick a genre to restrict the set to matching tracks. Leave it on Any to use your full library. Genres shown with ~ are fuzzy matches.

Added Time Filter

Limit the set to recently added tracks: Last 30 days, 90 days, 120 days, This year, or All.

Set Duration

Choose a target length (30–180 min). The generated set will not exceed it.

Playlist mode auto-selects duration: when you switch to From Playlist source mode, the closest duration pill that fits the playlist is selected automatically and all shorter ones are disabled.
06

Generate from playlist

By default, DJFriend draws from your Full Library. If you have imported playlists, a Source card appears above the Duration pills in the sidebar.

1
Toggle to "From Playlist"

Click the From Playlist button in the Source card. A dropdown appears listing all your imported playlists.

2
Pick a playlist

Select the playlist you want to work from. The duration pill auto-selects to the shortest option that fits the playlist's total length. Shorter pills are disabled.

3
Generate

Hit Generate. The set is built from that playlist only. Your other preferences (genre, BPM, curve) still apply.

This is the ideal workflow for a booked set: import the playlist you've prepared in Spotify, then let DJFriend find the best harmonic order within it.
07

Generate the set

Press the ▶ Generate button. DJFriend builds the full set automatically based on your curve, preferences, and library.

Get a different result

Click ▶ Generate again — each run produces a slightly different set. Use Generate from remaining to replace only the tracks after a specific position while keeping the ones before it.

08

Fit indicators

Every track in the generated set shows a small coloured dot in the # column. Hover over it to see exactly why a track might not fit.

Green — good fit. No issues with this track.

Amber — minor issue. Hover to see what's slightly off — could be energy, BPM, or genre.

Red — needs replacing. Hover to see the problem — harmonic clash, energy mismatch, or a big BPM jump.

The stats bar at the top shows how many tracks have issues. Click the count to jump to the first one.

09

Swapping tracks

Every track row has a ⟳ Swap button. Click it to open the smart swap panel.

DJFriend looks through your full library and shows up to 5 replacement tracks, sorted best to worst fit. If nothing suitable is found it says so. Click any suggestion to swap it in.

Manual reordering

Drag any track row to a new position using the grip handle on the left. The fit indicators update immediately to reflect the new order.

Removing a track

Click the ✕ Remove button on any row to delete it from the set. The track remains in your library — it is simply not part of this set.

10

Editing track metadata

Click the ✎ Edit button on any row to open the inline editor. You can update:

  • Title & Artist — change the display name.
  • BPM — manual override. Use ×2 / ÷2 to fix a halved or doubled reading in one click.
  • Key / Camelot — correct the key if it doesn't match what your DJ software shows.
  • Genre — add or change the genre.

Changes apply immediately and are saved for future sessions.

11

Export

Click the ↓ Export button at the top right of the set list to open the export menu.

M3U playlist

Exports an M3U file with absolute file paths. Open it directly in Rekordbox, Serato, VirtualDJ, Traktor, or any other DJ software that reads M3U. The file preserves track order and includes the runtime of each track.

Spotify playlist

Sends the set to your Spotify account as a new playlist. Requires Spotify credentials in Settings. Tracks that can't be found on Spotify are skipped.

Pro tip: use the Spotify export after a gig to save the actual set you played — import the playlist, generate from it in source mode, and you have a reproducible template for future bookings.
12

Settings

Open Settings via the gear icon in the top-right corner. There are three sections:

Spotify credentials

Enter your Spotify Client ID and Client Secret to enable genre data and Spotify export. Get them free at developer.spotify.com/dashboard — create an app and copy the credentials.

AI (Groq)

Enter your Groq API key to enable AI tagging (vibe, mood, venue fit). Groq is free at console.groq.com. Without it the app works fine — you just won't have AI tags.

Library path

Shows the currently loaded folder. Change it here to switch between libraries.
