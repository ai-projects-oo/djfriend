TC-A1 · Badge appears on existing history entries

TC-A1Badge renders for saved sets
Steps
  1. Open the app → click the History tab
  2. Look at the summary bar of any entry that has tracks
Expected
PASS — A small pill badge with a number (0–100) appears between the tag pills and the mini energy curve
PASS — Badge color is green, amber, or red based on the score value
PASS — Badge is visible without expanding the entry
If History is empty this test is N/A — create a set first (see prerequisite above).

TC-A2 · Tooltip shows component breakdown

TC-A2Hover tooltip displays score components
Steps
  1. Hover your mouse over the score badge on any history entry
Expected
PASS — A native browser tooltip appears with format: Harmonic: X% · Energy fit: Y% · BPM flow: Z%
PASS — All three values are between 0% and 100%

TC-A3 · Green badge — high-quality set

TC-A3Well-matched set scores green (≥80)
Steps
  1. Go to the Generator tab
  2. Set harmonic compatibility to maximum (e.g. pick a single starting key or let the algorithm choose)
  3. Generate a set — verify most/all tracks show no harmonic warning icons in the track list
  4. Export as M3U (this saves to History)
  5. Go to History → check the badge on this new entry
Expected
PASS — Badge is green and the number is ≥ 80
A well-generated set typically scores 85–95. If your library is small, scores may vary.

TC-A4 · Score reflects harmonic warnings

TC-A4Sets with many harmonic warnings score lower
Steps
  1. Generate a set — note which tracks show harmonic warning indicators in the tracklist
  2. Count the warnings (red/orange key indicators)
  3. Export as M3U → go to History → find the entry
  4. Hover the badge → read the Harmonic: X% value
Expected
PASS — Harmonic % in the tooltip is inversely proportional to the number of warning tracks you counted (more warnings = lower Harmonic %)
PASS — Overall score is lower than a set with no warnings

TC-A5 · Multiple entries — each badge is independent

TC-A5Each history entry shows its own score
Steps
  1. If you have 2+ entries in History, compare their badges side by side
  2. Verify that different sets show different scores (unless they happen to be identical)
Expected
PASS — Each entry has its own independent score, not the same value on every row
PASS — Score does not change when you collapse/expand the entry

TC-A6 · No badge for single-track entries

TC-A6Single-track or empty entries show no badge
Steps
  1. If you have a history entry with only 1 track, verify no badge appears
  2. (This is rare — most sets have multiple tracks. Skip if not applicable.)
Expected
PASS — No score badge rendered for 0- or 1-track entries

TC-A7 · Load entry does not affect score

TC-A7Loading a history entry into Generator doesn't corrupt score
Steps
  1. In History, click Load on any entry (loads the set back into Generator tab)
  2. Navigate back to the History tab
Expected
PASS — Score badge on all entries still shows the same values as before loading
PASS — No badge disappears or turns blank
