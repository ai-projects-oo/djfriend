What it does

  • Harmonic compatibility scoring (Camelot wheel)
  • Energy curve editor with 5 presets
  • BPM continuity scoring
  • Genre, vibe, mood, venue, and vocal filters
  • Set duration control — 30 to 180 minutes
  • Load from folder, Apple Music, M3U, or Rekordbox XML
  • Drag-to-reorder tracklist with harmonic warnings
  • One-click swap suggestions for any track
  • Export to M3U playlist or Spotify
  • Full set history — save, rename, reload, re-export
  • Spotify playlist checker — see which tracks you already own
